<task type="auto">
  <name>Task 2: SQLite schema + DB connection layer</name>
  <files>src/router/db.py</files>
  <action>
  1. Create `src/router/db.py` with:
     - `RouterDB` class wrapping sqlite3 connection
     - `__init__(db_path: str)`: open connection, enable WAL mode (`PRAGMA journal_mode=WAL`), enable foreign keys
     - `init_schema()`: CREATE TABLE IF NOT EXISTS for all 4 tables matching Pydantic models. Use TEXT for UUIDs, TEXT for ISO-8601 timestamps, TEXT for JSON payloads, INTEGER for attempt/priority/concurrency.
     - `tasks` table: UNIQUE on task_id, INDEX on status, INDEX on idempotency_key
     - `task_events` table: UNIQUE on idempotency_key (dedupe), INDEX on task_id, auto-increment event_id
     - `workers` table: UNIQUE on worker_id, INDEX on status
     - `leases` table: UNIQUE on task_id (one active lease per task), INDEX on expires_at
  2. Data access methods (all use `BEGIN IMMEDIATE` for writes):
     - `insert_task(task: Task) -> Task`
     - `get_task(task_id: str) -> Task | None`
     - `update_task_status(task_id: str, old_status: TaskStatus, new_status: TaskStatus) -> bool` (compare-and-set, returns False if old_status mismatch)
     - `insert_event(event: TaskEvent) -> bool` (returns False on duplicate idempotency_key)
     - `get_events(task_id: str) -> list[TaskEvent]`
     - `insert_worker(worker: Worker) -> Worker`
     - `get_worker(worker_id: str) -> Worker | None`
     - `list_workers(status: str | None) -> list[Worker]`
     - `create_lease(lease: Lease) -> Lease`
     - `get_active_lease(task_id: str) -> Lease | None`
     - `expire_lease(lease_id: str) -> bool`
  3. Handle `sqlite3.IntegrityError` for duplicate inserts gracefully.
  4. Handle `SQLITE_BUSY` with retry: 3 attempts, 50ms/100ms/200ms backoff.
  5. Schema invariant: `update_task_status` + lease operations in same transaction.
  </action>
  <verify>python -c "from src.router.db import RouterDB; db = RouterDB(':memory:'); db.init_schema(); print('Schema OK')"</verify>
